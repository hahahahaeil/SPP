跑步热折射中国民众健康观念变迁

“以前觉得跑步锻炼是中老年人干的事，我这么年轻，没必要去坚持锻炼。”25岁的宁夏银川市民王晓说。但现在，每天走路上班、每周长跑一次的他，已经离不开跑步了。

　　王晓的改变，始于3年前。当时身高不足1米7、体重接近80公斤的他在同事带动下开始跑步。“以前白天犯困，一直以为是没休息好，跑步后，作息不变却精神很多，后来才知道那叫亚健康。”王晓说，跑步逐渐成为他的一种生活状态。
　　在宁夏跑吧负责人尹洁看来，一些年轻人平时不注重锻炼，到中老年再“临时抱佛脚”的观念正在改变。“跑吧成员从最初不到20人，到如今的600多人，平均年龄也从40多岁为主逐渐向30岁过渡。”尹洁说，跑步是诸多运动方式之一，这项运动的火热折射出人们对健康生活方式的热情。
　　“坚持跑步能使一些负面情绪得以释放，可以说是身心双赢。”45岁的资深跑友武树军说，原计划退休后好好锻炼的他，在一次生病后“提前”加入跑步行列。如今，带领儿子和爱人一起跑步的他觉得“跑步要趁早，身体好了啥都好”。

跑步热不只出现在宁夏。

据媒体报道，2011年北京马拉松3万个名额仅6天就被抢完，而此前一年，这一名额耗时两月才被推销完，到2014年，10小时后再想报名的人已没有机会。

　　面对日益增多的跑步人群，一些地方政府也积极行动起来。宁夏银川市先后在当地的中山公园、海宝公园设立里程牌，方便跑步者计算距离，此前一些以体育场馆为中心分布的运动装备店，也在一些公园、绿地周边开设新店。记者在银川市一些公园观察发现，穿皮鞋悠闲散步的人少了，穿运动鞋慢跑或快走的人多了，“日行一万步，健康有好处”、“坚持到底，胜利归你”等标语也多了起来。
　　在兰州大学哲学社会学院社会学系教师台文泽博士看来，跑步热和居民健康观念转变的背后，是生活水平的提高。“健康意识属于形而上的部分，只有生活水平提高，物质基础有保障，人们才会有闲暇时间，也才会从享受生活的角度对身体健康更为关注，这样才有了跑步热的兴起。”台文泽说，随着经济社会发展，未来肯定有越来越多人加入到跑步等运动中来。
